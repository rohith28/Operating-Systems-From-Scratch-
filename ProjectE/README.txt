All the code is written in kernel.c file
To compile the file use compileOS.sh bash script
1)type ./compileOS.sh and execute
2)run the simulator jar
3)SHELL will be prompted in the output
4)type command execute phello
5)Hello world is printed continously
6)type any other shell command
7)Hello world still runs in the background, and the shell commands also get executed
8)Kill the process by command kill 1
9)Process gets killed and Shell is prompted back
